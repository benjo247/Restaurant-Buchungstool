# Restaurant App – Browser Only

Dieses Paket ist für GitHub + Vercel + Neon gedacht und braucht kein lokales Prisma.

## Dateien hochladen
Lade den gesamten Inhalt dieses Ordners in dein GitHub-Repo hoch.

## Vercel
- Repo in Vercel importieren
- Environment Variable `DATABASE_URL` setzen

## Neon
- SQL Editor öffnen
- Datei `neon-schema.sql` ausführen
